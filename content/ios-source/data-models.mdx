## Core Market Models

### Stock.swift <StatusBadge status="error">Core</StatusBadge>

**Location:** `Models/Stock.swift` (~120 lines)

The primary stock data model used throughout the application.

**Fields:**

| Property | Type | Description |
|----------|------|-------------|
| `id` | `String` | Unique identifier (stock code) |
| `rank` | `Int` | Ranking position in current filter |
| `name` | `String` | Company name (Korean or English) |
| `code` | `String` | Stock ticker code (e.g., "005930") |
| `exchange` | `StockExchange` | Exchange enum value |
| `marketCap` | `String` | Formatted market capitalization |
| `price` | `Int` | Current price in KRW or USD cents |
| `change` | `Int` | Price change from previous close |
| `changePercent` | `Double` | Change as percentage |
| `volume` | `Int64` | Trading volume |
| `dividendYield` | `Double` | Annual dividend yield percentage |
| `logoStyle` | `LogoStyle` | Logo rendering style |
| `isNXT` | `Bool` | Whether stock trades on NXT venue |
| `isFractional` | `Bool` | Whether fractional shares are available |

**Computed Properties:**

| Property | Returns | Description |
|----------|---------|-------------|
| `isPositive` | `Bool` | `change >= 0` |
| `tradingValue` | `Int64` | `price * volume` |
| `formattedPrice` | `String` | Locale-formatted price |
| `formattedChange` | `String` | Signed change with formatting |

**LogoStyle Enum:**

```swift
enum LogoStyle {
    case text(String, logoColor: Color)       // Full name: "SAMSUNG"
    case initials(String, logoColor: Color)   // Abbreviation: "H", "LG"
    case symbol(String, logoColor: Color)     // SF Symbol: "sun.max.fill"
}
```

**LogoStyleMapper:**

Static mapping class that returns the correct `LogoStyle` for known companies (Samsung, SK, Hyundai, LG, Apple, Tesla, etc.). Falls back to auto-generated initials for unknown stocks.

### StockDetailInfo.swift <StatusBadge status="error">Core</StatusBadge>

**Location:** `Models/StockDetailInfo.swift` (~200 lines)

The largest model in the app — comprehensive stock detail with 20+ nested types.

**Primary Sections:**

| Section | Fields | Description |
|---------|--------|-------------|
| Price | `prevClose`, `open`, `high`, `low` | OHLC price data |
| Fundamentals | `per`, `eps`, `pbr`, `bps`, `dividendPerShare` | Valuation metrics |
| Range | `week52High`, `week52Low` | 52-week trading range |
| Market | `tradingValueFormatted`, `marketCapFormatted`, `foreignOwnership` | Market stats |
| Chart | `chartData: [ChartPoint]` | Historical price data |
| Status | `marketStatus`, `nxtStatus` | Market session status |

**Extended Data Sections (Nested Types):**

| Type | Structure | Purpose |
|------|-----------|---------|
| `ChartPoint` | `date`, `open`, `high`, `low`, `close`, `volume` | Single candlestick data point |
| `InvestorFlow` | `days: [DayFlow]` with `foreign`, `individual`, `institution` | Net buying by investor type |
| `AnalystOpinion` | `buyCount`, `holdCount`, `sellCount`, `avgTargetPrice`, `opinions: [BrokerOpinion]` | Analyst consensus |
| `AnalystReport` | `title`, `broker`, `date`, `targetPrice`, `recommendation` | Individual research report |
| `OrderBook` | `asks: [Level]`, `bids: [Level]` (10 levels each) | Real-time order book |
| `DividendRecord` | `year`, `amount`, `yield`, `paymentDate` | Annual dividend record |
| `DividendYearGroup` | `year`, `quarters: [QuarterDividend]` | Quarterly dividend breakdown |
| `FinancialRecord` | `period`, `revenue`, `operatingProfit`, `netIncome` | Summary financials |
| `FinancialDetailRecord` | 15 metrics across quarterly/annual periods | Detailed financial metrics |

### MarketSessionStatus.swift

**Location:** `Models/MarketSessionStatus.swift` (~129 lines)

Enum managing market session state with countdown logic:

```swift
enum MarketSessionStatus {
    case preMarket(countdown: TimeInterval)
    case regularSession
    case nxtSession
    case postMarket
    case closed(nextOpen: Date?)
}
```

**Computed Properties:**
- `displayLabel` — Korean label ("장전", "거래중", "NXT 거래중", "장후", "장마감")
- `isLive` — Whether prices are streaming
- `currentStatus(for exchange:)` — Static method to compute current session based on KST time
