## Authentication Layer

### AuthManager.swift

<StatusBadge status="live">Core</StatusBadge>

`@Observable` singleton managing the entire authentication lifecycle.

**Auth States:**

```swift
enum AuthState {
    case unknown          // Initial state — checking stored token
    case loggedOut        // No valid session
    case loggedIn(User)   // Authenticated with user profile
}
```

**Key Methods:**

| Method | Description |
|--------|-------------|
| `checkStoredToken()` | Validates JWT from Keychain on app launch, fetches user profile |
| `login(email:password:)` | Email/password authentication → stores tokens → fetches user |
| `register(name:email:password:)` | Creates account → stores tokens → fetches user |
| `signInWithApple(identityToken:authCode:fullName:)` | Apple Sign In → stores tokens → fetches user |
| `refreshAccessToken()` | Silent token refresh using stored refresh token |
| `logout()` | Clears Keychain → sets state to `.loggedOut` |

**API Endpoints:**

| Method | Endpoint |
|--------|----------|
| Login | `POST /api/auth/login` |
| Register | `POST /api/auth/register` |
| Apple Sign In | `POST /api/auth/apple` |
| Get User | `GET /api/auth/me` |
| Refresh Token | `POST /api/auth/refresh` |

**Token Refresh Strategy:**

On 401 response, `APIClient` automatically attempts a token refresh. If refresh succeeds, the original request is retried. If refresh fails, user is logged out.

## Network Layer

### APIClient.swift

<StatusBadge status="live">Core</StatusBadge>

Lightweight HTTP client for REST API communication with the FastAPI backend.

```swift
struct APIClient: Sendable {
    let baseURL: URL
    // Simulator: http://127.0.0.1:8000
    // Device:    http://192.168.68.56:8000

    func get<T: Decodable>(_ path: String,
                           queryItems: [URLQueryItem]? = nil) async throws -> T

    func post<T: Decodable, U: Encodable>(_ path: String,
                                           body: U) async throws -> T
}
```

**Architecture:**
- Generic `get<T>()` and `post<T, U>()` methods handle all REST calls
- Automatic Bearer token injection from Keychain via `Authorization` header
- Automatic `snake_case` → `camelCase` JSON key conversion (decoding) and `camelCase` → `snake_case` (encoding)
- Comprehensive request/response logging for debugging
- Error handling via `APIError` enum with specific status code handling (401 unauthorized, 409 conflict, 422 validation)

**Error Types:**

```swift
enum APIError: LocalizedError {
    case invalidURL          // Malformed URL construction
    case invalidResponse     // Non-HTTPURLResponse
    case httpError(statusCode: Int)  // Non-2xx status code
}
```

**Base URL Detection:**

The client automatically selects the correct base URL based on the runtime environment:

```swift
#if targetEnvironment(simulator)
    let baseURL = URL(string: "http://127.0.0.1:8000")!
#else
    let baseURL = URL(string: "http://192.168.68.57:8000")!
#endif
```

> **Important for maintainers:** Update the device IP address when the development machine's local network address changes.

## WebSocket Layer

### WebSocketManager.swift

<StatusBadge status="live">Core</StatusBadge> <StatusBadge status="warning">Real-Time</StatusBadge>

Singleton manager for real-time stock price streaming via WebSocket.

**Connection Details:**

| Property | Value |
|----------|-------|
| URL | `ws://127.0.0.1:8000/ws/stream` (simulator) |
| Protocol | WebSocket via `URLSessionWebSocketTask` |
| Heartbeat | 30-second ping interval |
| Reconnection | Exponential backoff (1s → 2s → 4s → ... → 30s max) |
| Pattern | Singleton via `static let shared` |

**Key Properties:**

```swift
@Observable
final class WebSocketManager {
    static let shared = WebSocketManager()

    // Connection state
    var isConnected: Bool
    private var subscribedCodes: Set<String>

    // Per-code event streams
    private var continuations: [String: AsyncStream<PriceUpdate>.Continuation]
}
```

**Subscription Protocol:**

```json
// Subscribe to stock codes
{"action": "subscribe", "codes": ["005930", "000660"]}

// Unsubscribe
{"action": "unsubscribe", "codes": ["005930"]}

// Venue-aware subscription
{"action": "subscribe", "codes": ["005930"], "venue": "nxt"}
```

**Event Streaming:**

Each stock code gets its own `AsyncStream<PriceUpdate>` that ViewModels consume:

```swift
// In ViewModel
func startLiveUpdates() {
    let stream = WebSocketManager.shared.subscribe(code: stock.code)
    for await update in stream {
        latestPrice = update
    }
}
```

**Lifecycle:**
1. `connect()` — establishes WebSocket connection
2. `subscribe(codes:)` — sends subscribe message, creates AsyncStream
3. `receiveMessages()` — continuous loop decoding incoming PriceUpdate JSON
4. `unsubscribe(codes:)` — sends unsubscribe, cleans up continuations
5. `disconnect()` — closes connection gracefully
6. Auto-reconnect on unexpected disconnection

**Mock Mode:** When `useMocks` is true in `DependencyContainer`, the WebSocketManager's `disabled` flag prevents connection attempts.

### BAROColors.swift

<StatusBadge status="live">Design</StatusBadge>

|-------|-----|-------|

| `baroGold` | `#D4A017` | Secondary/accent |

**Semantic Colors:**

| Token | Purpose |
|-------|---------|
| `klText` / `klTextSecondary` / `klTextTertiary` | Text hierarchy |
| `klBackground` / `klSurface` / `klSurfaceSecondary` | Background layers |
| `klBorder` / `klDivider` | Borders and separators |
| `klPriceUp` (`#E04848` red) | Price increase |

## Extensions

### Color+Hex.swift

Adds hex color string initialization to SwiftUI `Color`:

```swift
Color(hex: "#E04848")    // With hash
Color(hex: "3478F6")     // Without hash
Color(hex: "FF3478F6")   // With alpha
```

### Double+Currency.swift

Number formatting extensions used throughout the app:

```swift
let price: Double = 188200
price.formattedWon        // "188,200"       — Korean won format
price.formattedIndex      // "188,200.00"    — Index value format
price.formattedPercent    // "0.02%"         — Percentage format
price.formattedVolume     // "188,200주"     — Volume with 주 suffix
price.formattedTradingValue // "5.4조" or "1,234억" — Smart unit selection
```

> **Note:** `formattedTradingValue` automatically selects 조 (trillion) or 억 (billion) units based on magnitude.

Korean-locale date formatting:

```swift
Date().koreanDateString    // "2025년 3월 14일 금요일"
Date().koreanShortDate     // "3/14"
```
