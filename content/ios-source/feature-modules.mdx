**Key Behaviors:**

**Data Loading Flow:**

```swift
func loadAllData() async {
    async let indices = loadIndices()
    async let stocks = loadStocks()
    async let banners = loadBanners()
    async let news = loadNews()
    async let rates = loadCurrencyRates()
    async let themes = loadThemes()
    _ = await (indices, stocks, banners, news, rates, themes)
}
```

**WebSocket Integration:**

```swift
func startLiveUpdates() async {
    for code in stocks.map(\.code) {
        let stream = WebSocketManager.shared.subscribe(code: code)
        for await update in stream {
            latestPrices[update.code] = update
        }
    }
}
```

### StockDetailPageViewModel.swift

**Location:** `Features/StockDetailPage/StockDetailPageViewModel.swift`

The most complex ViewModel in the application.

**Key Responsibilities:**

| Responsibility | Implementation |
|---------------|----------------|
| Data loading | `loadDetail()` тЖТ fetches `StockDetailInfo` from repository |
| Venue switching | Tracks `selectedVenue` (KRX/NXT/Combined), reloads on change |
| Chart management | `rebuildChartCache()` тАФ computes indicators, caches visible points |
| Live prices | `startLiveUpdates()` тАФ subscribes to WebSocket stream |
| Period selection | Fetches new chart data when period changes (1D, 1M, 3M, 1Y, 3Y) |
| Price scrubbing | `scrubbedIndex` тАФ tracks finger position on chart |
| Indicator config | `ChartSettings` management, indicator visibility toggles |

**Chart Cache System:**

When chart data or settings change, `rebuildChartCache()` recomputes:
1. Visible data points based on zoom/pan
2. Moving averages (SMA, EMA) with configurable periods
3. Bollinger Bands (upper, middle, lower)
4. RSI, MACD, and other sub-indicators
5. Price range for Y-axis scaling
