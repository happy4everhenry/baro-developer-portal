Complete directory layout and file inventory of the BARO iOS application.

## Root Directory

```
ios/BARO/
├── BARO/                          # Main app target
│   ├── BAROApp.swift              # @main entry point
│   ├── BARO-Info.plist            # App configuration
│   ├── App/                       # Application configuration
│   │   └── DependencyContainer.swift
│   ├── Core/                      # Shared infrastructure
│   │   ├── Auth/                  # Authentication (AuthManager, KeychainHelper)
│   │   ├── Network/               # HTTP client
│   │   ├── WebSocket/             # Real-time streaming
│   │   ├── DesignSystem/          # Colors, fonts, spacing, animations
│   │   ├── Extensions/            # Swift type extensions
│   │   ├── FavoritesManager.swift # Favorites state management
│   │   └── Utilities/             # Keyboard prewarmer, etc.
│   ├── Models/                    # Domain data models (19 files)
│   ├── Repositories/              # Data access layer
│   │   ├── Protocols/             # Repository interfaces (4 files)
│   │   ├── API/                   # Live API implementations
│   │   └── Mock/                  # Mock data implementations
│   ├── Features/                  # Feature modules (screens)
│   │   ├── MainTabView.swift      # Root tab navigation
│   │   ├── Splash/                # Splash screen with animated progress
│   │   ├── Auth/                  # Login + Register screens
│   │   ├── Home/                  # Home screen + 18 components
│   │   ├── Search/                # Stock search
│   │   ├── StockDetailPage/       # Stock detail (primary)
│   │   ├── StockDetail/           # Stock detail (legacy)
│   │   ├── NewsDetail/            # News article view
│   │   ├── ThemeDetail/           # Market theme view
│   │   ├── MarketOverview/        # Market indices detail
│   │   ├── AIInsights/            # AI-powered insights hub
│   │   ├── MyPage/                # Profile, settings, services
│   │   ├── Events/                # Promotions and competitions
│   │   ├── Notifications/         # Notification feed + settings
│   │   └── Placeholders/          # Tab placeholder screens
│   └── Components/                # Reusable UI components (10 files)
├── BAROTests/                     # Unit tests
└── BAROUITests/                   # UI tests
```

## Naming Conventions

| Pattern | Convention | Example |
|---------|-----------|---------|
| Views | `XyzView.swift` | `HomeView.swift`, `SearchView.swift` |
| ViewModels | `XyzViewModel.swift` | `HomeViewModel.swift` |
| Components | Descriptive name | `StockRowView.swift`, `BannerCarouselView.swift` |
| Design System | `KL` or `BARO` prefix | `KLSegmentedControl.swift`, `BAROColors.swift` |
| Extensions | `Type+Feature.swift` | `Color+Hex.swift`, `Double+Currency.swift` |
| Protocols | `XyzProtocol.swift` | `MarketRepositoryProtocol.swift` |
| Mock impl | `MockXyz.swift` | `MockMarketRepository.swift` |
| API impl | `APIXyz.swift` | `APIMarketRepository.swift` |
| Sheet/Modal | `XyzSheet.swift` | `StockFilterSheet.swift` |
