<InfoBox type="tip" title="Think of fonts like a team">
Each font token has a specific job. **Titles** are the team captains (big and bold). **Body text** does the everyday work (medium size). **Captions** handle the small details (compact and quiet). Just like a team, they work best when everyone stays in their role.
</InfoBox>

## Font Stack

BARO uses system fonts for fast rendering and a native feel. No custom font files are bundled.

| Platform | Latin / Numbers | Korean | Fallback |
|----------|----------------|--------|----------|
| **iOS / iPadOS** | `SF Pro Text` / `SF Pro Display` | Apple SD Gothic Neo | system-ui |
| **Android** | `Roboto` | Noto Sans KR | sans-serif |
| **Web (dev-docs)** | `Inter` | Noto Sans KR | system-ui, -apple-system |

<InfoBox type="note" title="SF Pro Licensing">
SF Pro is available only on Apple platforms. `Font.system()` in SwiftUI automatically resolves to SF Pro. Never embed SF Pro font files in the app bundle.
</InfoBox>

## Complete Type Scale

All 12 tokens from largest to smallest. Weights range from Regular (400) through Medium (500), Semibold (600), to Bold (700).

| Token | Size | Weight | Line Height | Usage |
|-------|------|--------|-------------|-------|
| `klTitle` | `20pt` | Bold (700) | ~24pt | Section titles, date headers |
| `klBannerTitle` | `18pt` | Bold (700) | ~22pt | Banner card headlines |
| `klHeadline` | `17pt` | Bold (700) | ~22pt | Stock section headers, index values |
| `klPrice` | `16pt` | Bold (700) | ~20pt | Stock prices, primary numeric data |
| `klSubheadline` | `15pt` | Semibold (600) | ~20pt | Company names, row titles |
| `klBody` | `14pt` | Semibold (600) | ~18pt | Filter tabs, active body text |
| `klBodyRegular` | `14pt` | Regular (400) | ~18pt | Banner subtitles, news titles |
| `klIndexName` | `13pt` | Semibold (600) | ~18pt | Market index names |
| `klCaption` | `12pt` | Regular (400) | ~16pt | Market cap, tags, captions |
| `klCaptionBold` | `12pt` | Semibold (600) | ~16pt | Price change values |
| `klSmall` | `11pt` | Semibold (600) | ~14pt | Index changes, market status |
| `klNavLabel` | `11pt` | Medium (500) | ~14pt | Bottom navigation labels |

## Contextual Pairings

Tokens work in pairs within UI components. These are the standard pairings used across BARO.

### Stock List Row
- Company name: `klSubheadline` (15pt Semibold) + ticker: `klCaption` (12pt Regular)
- Price: `klPrice` (16pt Bold) + change: `klCaptionBold` (12pt Semibold)

### Market Index Card
- Index name: `klIndexName` (13pt Semibold) + value: `klHeadline` (17pt Bold) + change: `klSmall` (11pt Semibold)

## Swift Implementation

All font tokens are static properties on `Font`. No custom font files needed -- system font is used throughout.

```swift filename="KLINEFonts.swift"
import SwiftUI

extension Font {
    /// 20pt bold -- Section titles, date display
    static let klTitle = Font.system(size: 20, weight: .bold)

    /// 18pt bold -- Banner title
    static let klBannerTitle = Font.system(size: 18, weight: .bold)

    /// 17pt bold -- Stock section header, index values
    static let klHeadline = Font.system(size: 17, weight: .bold)

    /// 16pt bold -- Stock prices
    static let klPrice = Font.system(size: 16, weight: .bold)

    /// 15pt semibold -- Company names
    static let klSubheadline = Font.system(size: 15, weight: .semibold)

    /// 14pt semibold -- Filter tabs, body text
    static let klBody = Font.system(size: 14, weight: .semibold)

    /// 14pt regular -- Banner subtitle, news title
    static let klBodyRegular = Font.system(size: 14, weight: .regular)

    /// 13pt semibold -- Index names
    static let klIndexName = Font.system(size: 13, weight: .semibold)

    /// 12pt regular -- Market cap, tags, captions
    static let klCaption = Font.system(size: 12, weight: .regular)

    /// 12pt semibold -- Price change values
    static let klCaptionBold = Font.system(size: 12, weight: .semibold)

    /// 11pt semibold -- Index change, market status
    static let klSmall = Font.system(size: 11, weight: .semibold)

    /// 11pt medium -- Navigation labels
    static let klNavLabel = Font.system(size: 11, weight: .medium)
}
```

```swift filename="Usage Example"
// Stock row view
Text(stock.name)
    .font(.klSubheadline)
    .foregroundColor(.klText)

Text(stock.formattedPrice)
    .font(.klPrice)
    .foregroundColor(.klText)

Text(stock.changePercent)
    .font(.klCaptionBold)
    .foregroundColor(stock.isUp ? .klPriceUp : .klPriceDown)
```
