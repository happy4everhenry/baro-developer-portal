<AccordionItem title="BannerCarouselView" icon="🎠" badge="Banner">

Auto-playing banner carousel using SwiftUI `TabView` with page-style swiping. Includes dot indicators and a play/pause toggle.

<InfoBox type="tip" title="Auto-play timing">
Banners advance every 3 seconds with a smooth 0.4s easeInOut transition. The dots at the bottom show which banner you are on!
</InfoBox>

<PropsTable>
| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `banners` | `[BannerItem]` | Required | Array of banner data with title, subtitle, icon, gradientColors |
</PropsTable>

| Property | Value |
|----------|-------|
| Timer interval | 3.0 seconds |
| Transition animation | `.easeInOut(duration: 0.4)` |
| Dot indicator size | 6pt circles |
| Active dot color | `#6B7280` |
| Inactive dot color | `#D1D5DB` |
| Banner height | 110pt (container) / 100pt (card) |
| Corner radius | `radiusXLarge` (24pt) |

```swift filename="BannerCarouselView.swift"
private let timer = Timer.publish(every: 3.0, on: .main, in: .common).autoconnect()

.onReceive(timer) { _ in
    guard isPlaying, banners.count > 1 else { return }
    withAnimation(.easeInOut(duration: 0.4)) {
        currentPage = (currentPage + 1) % banners.count
    }
}
```

<AccordionItem title="MarketThemeBubbleChartView" icon="🫧" badge="Themes">

A force-directed packed circle chart that shows market themes as bubbles. Bigger bubbles = bigger themes. Uses a spiral placement algorithm so bubbles do not overlap. Each bubble is tappable!

<PropsTable>
| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `themes` | `[MarketTheme]` | Required | Theme data with name, weight (0-1), bubbleColor |
| `onTap` | `(MarketTheme) -> Void` | Required | Callback when a bubble is tapped |
</PropsTable>

### Layout Algorithm

| Parameter | Value | Description |
|-----------|-------|-------------|
| Min radius | 16pt | Smallest possible bubble |
| Max radius | 60pt | Largest possible bubble |
| Padding between bubbles | 3pt | Minimum gap |
| Radius formula | `min + (max - min) * sqrt(weight)` | Square root scaling for perceptual accuracy |
| Aspect ratio | 1.1 | Slightly wider than tall |

### Font Sizing

| Radius Range | Font Size |
|-------------|-----------|
| >= 50pt | 16pt |
| >= 35pt | 13pt |
| >= 25pt | 11pt |
| < 25pt | 9pt |

```swift filename="MarketThemeBubbleChartView.swift"
// Sorted largest-first, spiral outward from center
let sorted = themes.sorted { $0.weight > $1.weight }

for theme in sorted {
    let r = radius(for: theme)
    let pos = findPosition(for: r, among: placed, center: center, bounds: size)
    placed.append((theme.id, pos, r))
}

// Spiral search: incrementally increase angle and distance
// until a non-overlapping position is found
var angle: CGFloat = 0
var dist: CGFloat = 0
let step: CGFloat = 2
let angleStep: CGFloat = 0.3

while dist < max(bounds.width, bounds.height) {
    let x = center.x + cos(angle) * dist
    let y = center.y + sin(angle) * dist
    angle += angleStep
    dist += step / (2 * .pi)  // gradually spiral outward
}
```

## Row & List Components

These components display data in horizontal rows and scrollable lists.

<Accordion>
<AccordionItem title="StockRowView" icon="📈" badge="Stock" defaultOpen={true}>

A single stock list row with rank, company logo, name, subtitle (varies by filter), price, and price change. Supports real-time WebSocket price overlay with animated digit transitions via `.numericText()`.

<PropsTable>
| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `stock` | `Stock` | Required | Full stock model with rank, name, price, logo style, etc. |
| `filter` | `StockFilter?` | `nil` | Determines which subtitle to display below the company name |
| `livePrice` | `PriceUpdate?` | `nil` | Optional WebSocket real-time price overlay |
</PropsTable>

### Subtitle Variants by Filter

| Filter | Subtitle | Example |
|--------|----------|---------|
| `.marketCap` / `nil` | Market capitalization | <KoreanLabel>시가총액</KoreanLabel> 1,114.08조 |
| `.volume` | Trading volume | <KoreanLabel>거래량</KoreanLabel> 28,540,000 |
| `.popular` | Trading value | <KoreanLabel>거래대금</KoreanLabel> 5.3조 |
| `.dividend` | Dividend yield | <KoreanLabel>배당수익률</KoreanLabel> 2.35% |
| `.gainers` / `.losers` | No subtitle | -- |

```swift filename="StockRowView.swift"
// Display values: prefer live data over REST snapshot
private var displayPrice: Int { livePrice?.price ?? stock.price }
private var displayChange: Int { livePrice?.change ?? stock.change }
private var displayChangePercent: Double { livePrice?.changePercent ?? stock.changePercent }

// Animated digit transition for price
Text(displayPrice.formattedWon)
    .font(.klPrice)
    .contentTransition(.numericText())
    .animation(.easeInOut(duration: 0.3), value: displayPrice)
```
