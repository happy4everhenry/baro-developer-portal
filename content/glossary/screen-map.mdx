# 화면별 용어 맵

<PageMeta status="live" layer="shared" />

앱 화면에서 보이는 금융 용어를 클릭하면 해당 개념의 **상세 설명 페이지**로 이동합니다. 금융 배경 지식이 없는 개발자가 화면을 보면서 "이게 뭐지?" 할 때 바로 찾을 수 있도록 구성했습니다.

---

## BARO 전체 화면 목록 (14 Screens)

BARO iOS 앱의 모든 화면과 각 화면이 연결되는 백엔드 서비스를 한눈에 볼 수 있는 인벤토리입니다.

| # | Screen | iOS Source Path | Status | Backend Services |
|---|--------|----------------|--------|-----------------|
| 1 | **Splash** | `Features/Splash/SplashView.swift` | <StatusBadge status="live">Live</StatusBadge> | AuthManager (token validation) |
| 2 | **Login / Register** | `Features/Auth/LoginView.swift`, `RegisterView.swift` | <StatusBadge status="live">Live</StatusBadge> | Auth API (JWT), Apple Sign In, Supabase Auth |
| 3 | **Home** | `Features/Home/HomeView.swift` | <StatusBadge status="live">Live</StatusBadge> | Stock API, Market Index API, Currency API, News API, Theme API, KIS WebSocket (live prices) |
| 4 | **Stock Detail** | `Features/StockDetailPage/StockDetailPageView.swift` | <StatusBadge status="live">Live</StatusBadge> | Stock Detail API, Chart API, OrderBook API, Investor Flow API, Analyst Opinion API, Financial API, Dividend API, KIS WebSocket |

<InfoBox type="info" title="Navigation Structure">
`MainTabView` is the root container with 5 tabs: Home, Products (placeholder), Assets (placeholder), Events, and Menu (MyPage). Other screens are reached via navigation stack pushes or modal sheets. The Search screen is presented as a full-screen sheet from the Home tab.
</InfoBox>
