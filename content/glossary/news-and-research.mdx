# 뉴스 및 리서치

<PageMeta status="live" layer="shared" />

증권사 리서치 리포트, 공시, 투자의견 등 정보 소스에 대한 설명입니다. 앱의 `AnalystOpinionDetailView`, `ReportDetailView`, `NewsDetailView`에서 표시됩니다.

---

## 컨텐츠 유형

앱에서 제공하는 금융 정보는 3가지 유형으로 분류됩니다:

| 유형 | 한국어 | 설명 | 신뢰도 |
|------|--------|------|--------|
| **뉴스** | 뉴스 | 언론사의 경제/시장 기사 | 언론사에 따라 다름 |
| **리포트** | 리포트 | 증권사 애널리스트가 작성한 분석 보고서 | 높음 (전문 분석) |
| **공시** | 공시 | 회사가 법적 의무에 따라 발표한 공식 문서 | 매우 높음 (공식 정보) |

```swift
// NewsContentType enum
enum NewsContentType: String, CaseIterable {
    case all         // "전체"
    case news        // "뉴스"
    case report      // "리포트"
    case disclosure  // "공시"
}
```

## 뉴스 (News)

경제, 산업, 기업 관련 언론 보도입니다.

### 뉴스 카테고리

| 카테고리 | 한국어 | 영향 범위 |
|----------|--------|-----------|
| 경제 | 경제 | 시장 전체 (금리, GDP, 물가 등) |
| 산업 | 산업 | 특정 업종 (반도체, 자동차 등) |
| 기술 | 기술 | IT/기술 섹터 |
| 에너지 | 에너지 | 에너지/원자재 섹터 |
| 국제 | 국제 | 해외 시장 동향 |

### 뉴스 필터

앱에서 `NewsFilterSheet`를 통해 2가지 축으로 필터링:

- **지역 (Region):** 전체 / 국내 / 해외
- **카테고리 (Category):** 전체 / 시장 / 종목 / 테마 / 경제 / 국제

### AI 요약

일부 기사에는 **AI가 생성한 요약**이 제공됩니다. `NewsDetailView`의 `expandedSummaryId` 상태로 토글됩니다.

## 투자의견 (Analyst Opinion)

증권사 애널리스트들의 **종목에 대한 평가**입니다. `AnalystOpinionDetailView`에서 표시됩니다.

### 투자의견 종류

| 의견 | 한국어 | 의미 | 앱 색상 |
|------|--------|------|---------|
| **Buy (매수)** | 매수 | "이 주식을 사라" | 🟢 초록 |
| **Hold (중립)** | 중립 | "보유하되 추가 매수는 보류" | ⚪ 회색 |
| **Sell (매도)** | 매도 | "이 주식을 팔아라" | 🔴 빨간 |

> **실제 관행:** 증권사들은 **매도 의견을 거의 내지 않습니다** (해당 기업과의 관계 때문). 따라서 "중립" 의견도 실질적으로는 부정적인 신호일 수 있습니다.

### 컨센서스 (Consensus)

여러 증권사 의견을 종합한 **전체 평가**입니다.

```
예시:
  매수 12명 | 중립 3명 | 매도 0명
  ████████████████████████░░░░░░  (80% 매수)

  컨센서스 목표주가: 85,000원
  현재가: 72,300원
  괴리율: +17.6% (상승 여력)
```

앱에서는 `GeometryReader`로 비율 막대를 그립니다. 각 의견의 수에 비례하여 색상 블록의 너비가 결정됩니다.

### 목표주가 (Target Price)

애널리스트가 **향후 12개월 내 도달할 것으로 예상하는 가격**입니다.

- 목표주가 > 현재가 → 상승 여력이 있다는 의미
- 목표주가 < 현재가 → 고평가 상태라는 의미

```
괴리율 = (목표주가 - 현재가) / 현재가 × 100%
```

## 공시 (Disclosure / DART)

회사가 **법적 의무에 따라 공개하는 공식 정보**입니다.

### 공시의 중요성

공시는 **가장 신뢰할 수 있는 정보 소스**입니다. 금융감독원의 DART (전자공시시스템)를 통해 공개됩니다.

### 주요 공시 유형

| 공시 유형 | 설명 | 주가 영향 |
|-----------|------|-----------|
| **실적 공시** | 분기/연간 실적 발표 | 매우 큼 |
| **배당 공시** | 배당금 결정 공지 | 큼 |
| **유상증자** | 새 주식 발행 (자금 조달) | 부정적 (주식 가치 희석) |
| **자사주 매입** | 회사가 자기 주식을 사들임 | 긍정적 (주가 지지) |
| **대표이사 변경** | 경영진 교체 | 상황에 따라 다름 |
| **합병/인수** | M&A 관련 공시 | 큼 (호재 또는 악재) |
| **소송/제재** | 법적 분쟁 관련 | 부정적 |

> **개발자 참고:** 공시는 장 종료 후에 나오는 경우가 많습니다. 장 중에 공시가 나오면 주가가 급변할 수 있어, 실시간 공시 알림이 중요합니다.

## 뉴스 강도 (News Intensity)

앱의 시장 지표에서 사용되는 개념:

| 지표 | 설명 |
|------|------|
| **뉴스건수** | 특정 종목에 대한 뉴스 기사 수 |
| **공시건수** | 특정 종목의 공시 건수 |
| **뉴스강도** | 뉴스 빈도 × 관련성 가중치 |

뉴스가 갑자기 많아지면 → 해당 종목에 중요한 이벤트가 발생했거나 발생 중
