# 거래소 및 마켓

<PageMeta status="live" layer="shared" />

주식이 거래되는 장소(거래소)와 거래 방식(venue)에 대한 설명입니다. 앱의 `StockExchange`, `MarketVenue`, `MarketSessionStatus` 모델과 관련됩니다.

## 한국 주식시장 구조

한국 주식시장은 **한국거래소(KRX)**가 운영하며, 두 개의 시장으로 나뉩니다.

### KOSPI (코스피)

**K**orea Composite **S**tock **P**rice **I**ndex의 약자입니다.

- 한국을 대표하는 대형주 시장
- 상장 요건이 엄격 (자본금, 매출, 이익 기준 충족 필요)
- 대표 종목: 삼성전자, SK하이닉스, 현대차, LG에너지솔루션
- 약 800개 종목 상장
- 시장 지수: 코스피 지수 (예: 2,723.15)

### KOSDAQ (코스닥)

**K**orean **S**ecurities **D**ealers **A**utomated **Q**uotations의 약자입니다.

- 기술주, 바이오, 중소형 성장주 중심 시장
- 미국의 NASDAQ과 유사한 성격
- 상장 요건이 KOSPI보다 낮음 (성장 잠재력 중시)
- 대표 종목: 셀트리온, 카카오게임즈, 에코프로비엠
- 변동성이 KOSPI보다 큰 경향

### KRX vs NXT

| 항목 | KRX (정규장) | NXT (넥스트레이드) |
|------|-------------|-------------------|
| **정식 명칭** | 한국거래소 | 넥스트레이드 (대체거래소) |
| **운영 시간** | 09:00 ~ 15:30 | 08:00 ~ 20:00 |
| **데이터** | 실시간 | 실시간 |
| **거래 가능 종목** | 모든 상장 종목 | KOSPI 일부 종목 |
| **특징** | 메인 시장 | 정규장 외 시간에도 거래 가능 |

```swift
// MarketVenue.swift — 앱에서의 venue 구분
enum MarketVenue: String, CaseIterable {
    case combined   // "통합" — KRX + NXT 합산
    case krx        // "정규장" — KRX만
    case nxt        // "NXT" — NXT만
}
```

> **개발자 참고:** 같은 종목이라도 KRX와 NXT에서 가격이 다를 수 있습니다. `StockPriceHeaderView`의 venue selector에서 사용자가 어떤 시세를 볼지 선택합니다.

## 미국 주식시장

앱에서 지원하는 미국 거래소:

### NASDAQ (나스닥)

- 세계 최대 전자 거래소
- 기술 대형주 중심: Apple, Microsoft, Google, Amazon, Tesla
- 거래 시간: 09:30~16:00 (미국 동부 시간) → 한국 시간 23:30~06:00

### NYSE (뉴욕증권거래소)

- 세계 최대 규모 거래소 (시가총액 기준)
- 전통 대형주 중심: JPMorgan, Walmart, Johnson & Johnson
- Wall Street에 위치한 물리적 거래소

### AMEX (아멕스)

- American Stock Exchange (현재 NYSE American)
- 중소형주와 ETF 중심

> **개발자 참고:** 미국 주식 데이터는 **15분 지연**입니다. 이는 실시간 데이터 라이센스 비용 때문이며, 앱에서 미국 주식 지수의 상태를 표시할 때 이 점을 반영합니다.

## 시장 운영 시간

### 한국 시장 타임라인

```
08:00       09:00              15:20   15:30       20:00
  |           |                  |       |           |
  |← NXT 프리→|←── KRX 정규장 ──→|→ 마감 →|← NXT 야간→|
              |←── NXT 동시 운영 ─────────→|
```

### 시장 세션 상태

`MarketSessionStatus` enum으로 관리됩니다:

| 상태 | 한국어 표시 | 앱 UI | 설명 |
|------|-----------|-------|------|
| `.trading` | 거래중 | 🟢 녹색 + 맥동 애니메이션 | 정규장 거래 시간 |
| `.preMarket` | 장전 | 🟡 노란색 | 장 시작 전 |
| `.closed` | 장종료 | ⚪ 회색 | 거래 종료 |

```swift
// MarketSessionStatus.swift
static var current: MarketSessionStatus {
    // 현재 시간에 따라 자동 결정
    // 09:00~15:30 → .trading
    // 그 외 → .closed 또는 .preMarket
}
```

## 환율 (Exchange Rate)

앱의 `CurrencyRate` 모델과 `MarketOverviewDetailView`에서 표시됩니다.

### 표시 방식

한국에서 환율은 **1 외국 통화 = XX 원** 형식으로 표시합니다.

| 통화 | 기준 단위 | 예시 | 이유 |
|------|-----------|------|------|
| USD (달러) | 1 달러 | 1,481.60원 | 기본 단위 |
| EUR (유로) | 1 유로 | 1,612.30원 | 기본 단위 |
| JPY (엔) | **100 엔** | 964.20원 | 1엔 ≈ 9원이라 100엔 기준 사용 |
| CNY (위안) | 1 위안 | 203.50원 | 기본 단위 |

> **개발자 참고:** `CurrencyRate.unitLabel`이 통화마다 다릅니다. JPY는 "100 JPY", 나머지는 "1 USD" 등. 이것은 금융 관례입니다.

### 환율과 주식의 관계

- 원화 약세 (환율 상승) → 수출 기업에 유리 (삼성전자, 현대차 등)
- 원화 강세 (환율 하락) → 수입 기업에 유리, 외국인 투자 유입 가능
- 미국 금리 인상 → 달러 강세 → 원화 약세 → 외국인 자금 유출 가능

## 종목 정렬 필터

앱의 `StockFilter` enum으로 정의된 정렬 기준:

| 필터 | 한국어 | 설명 |
|------|--------|------|
| `.marketCap` | 시가총액 | 회사 규모 순 |
| `.popular` | 인기거래 | 관심/거래 활발도 순 |
| `.volume` | 거래량 | 거래된 주식 수 순 |
| `.gainers` | 상승 | 오늘 상승률 높은 순 |
| `.losers` | 하락 | 오늘 하락률 높은 순 |
| `.dividend` | 배당 | 배당수익률 높은 순 |
