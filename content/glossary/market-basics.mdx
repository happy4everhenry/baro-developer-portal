# 주식시장 기초

<PageMeta status="live" layer="shared" />

주식 거래 앱을 유지보수하기 위해 알아야 할 주식시장의 기본 개념입니다.

---

## 주식(Stock)이란?

주식은 **회사의 소유권 일부**입니다. 삼성전자 주식 1주를 사면, 삼성전자라는 회사의 아주 작은 지분을 소유하게 됩니다.

- **종목 (Stock/Security):** 거래 가능한 하나의 주식을 말합니다. 앱에서 `Stock` 모델이 이에 해당합니다.
- **종목코드 (Stock Code):** 각 종목을 식별하는 고유 번호. 예: 삼성전자 = `005930`, SK하이닉스 = `000660`

```swift
// Stock.swift에서
struct Stock {
    let code: String      // "005930" — 종목코드
    let name: String      // "삼성전자" — 종목명
    let price: Int        // 현재가 (원 단위)
    let change: Int       // 전일 대비 변동액
    let changePercent: Double  // 전일 대비 변동률 (%)
}
```

## 주가(Stock Price)는 어떻게 결정되나?

주가는 **사려는 사람(매수자)**과 **팔려는 사람(매도자)** 사이의 합의 가격입니다.

- 사려는 사람이 많으면 → 가격 **상승** 📈
- 팔려는 사람이 많으면 → 가격 **하락** 📉

이 과정은 **호가창(Order Book)**에서 실시간으로 일어납니다. 자세한 내용은 [호가 및 매매](/financial-glossary/order-book-and-trading) 참조.

## 장(Market Session) — 시장 운영 시간

주식시장은 24시간 열려있지 않습니다. 정해진 시간에만 거래할 수 있습니다.

### KRX (한국거래소) 정규장

| 시간 | 상태 | 앱 표시 | `MarketSessionStatus` |
|------|------|---------|----------------------|
| 09:00 ~ 15:30 | 정규 거래 시간 | 🟢 거래중 | `.trading` |
| 15:30 이후 | 장 마감 | ⚪ 장종료 | `.closed` |
| 09:00 이전 | 장 시작 전 | 🟡 장전 | `.preMarket` |

### NXT (넥스트레이드) — 시간외 거래

NXT는 한국의 **대체거래소(ATS, Alternative Trading System)**입니다. 정규장 종료 후에도 거래할 수 있습니다.

| 시간 | 설명 |
|------|------|
| 08:00 ~ 09:00 | NXT 프리마켓 |
| 09:00 ~ 15:30 | NXT + KRX 동시 운영 |
| 15:40 ~ 20:00 | NXT 단독 (시간외 거래) |

```swift
// MarketVenue.swift에서
enum MarketVenue: String, CaseIterable {
    case combined   // "통합" — KRX + NXT 합산 시세
    case krx        // "정규장" — KRX만의 시세
    case nxt        // "NXT" — NXT만의 시세
}
```

> **개발자 참고:** 앱의 `StockPriceHeaderView`에서 사용자가 시세를 전환할 수 있습니다 (통합/정규장/NXT). 각 venue별로 가격이 다를 수 있습니다.

## 거래소(Exchange)

거래소는 주식을 사고파는 **시장**입니다. 앱은 한국과 미국 거래소를 지원합니다.

### 한국 거래소

| 거래소 | 설명 | 특징 |
|--------|------|------|
| **KOSPI (코스피)** | 한국 종합주가지수 시장 | 대형주 중심 (삼성전자, 현대차 등). 약 800개 종목 |
| **KOSDAQ (코스닥)** | 기술주 중심 시장 | 중소형 기술/바이오 기업 중심. NASDAQ과 비슷한 성격 |

### 미국 거래소

| 거래소 | 설명 | 특징 |
|--------|------|------|
| **NASDAQ (나스닥)** | 미국 기술주 시장 | Apple, Google, Amazon 등 기술 대형주 |
| **NYSE (뉴욕증권거래소)** | 미국 최대 거래소 | 전통 대형주 중심 |
| **AMEX (아멕스)** | American Stock Exchange | 중소형주, ETF 중심 |

```swift
// StockExchange.swift에서
enum StockExchange: String, CaseIterable {
    case kospi   // 🇰🇷 코스피
    case kosdaq  // 🇰🇷 코스닥
    case nasdaq  // 🇺🇸 나스닥
    case nyse    // 🇺🇸 뉴욕
    case amex    // 🇺🇸 아멕스
}
```

> **개발자 참고:** 한국 주식은 **실시간(WebSocket)** 데이터를 사용하고, 미국 주식은 **15분 지연** 데이터를 사용합니다. 이는 데이터 라이센스 비용 때문입니다.

## 지수(Market Index)

지수는 여러 종목의 가격을 종합해서 **시장 전체의 방향**을 보여주는 숫자입니다.

| 지수 | 설명 | 예시 |
|------|------|------|
| **코스피 지수** | KOSPI 시장에 상장된 모든 종목의 시가총액 가중 평균 | 2,723.15 (+0.46%) |
| **코스닥 지수** | KOSDAQ 시장 종합 지수 | 845.20 (-0.12%) |
| **나스닥 지수** | NASDAQ 시장 종합 지수 | 17,808.25 (+1.2%) |
| **S&P 500** | 미국 대형주 500개 종합 지수 | 5,667.56 (+0.8%) |

> **비유:** 지수는 "시장의 체온"과 같습니다. 지수가 오르면 시장 전체가 좋은 것이고, 내리면 시장이 안 좋은 것입니다.

앱에서는 `MarketIndex` 모델과 `MarketOverviewRowView`에서 지수를 표시합니다.

## 환율(Exchange Rate)

환율은 **한 나라의 화폐를 다른 나라 화폐로 교환할 때의 비율**입니다.

예: USD/KRW = 1,481.60 → 1달러 = 1,481.60원

| 표시 | 의미 |
|------|------|
| USD/KRW 1,481.60 | 1달러 = 1,481.60원 |
| JPY/KRW 964.20 | 100엔 = 964.20원 (엔화는 100단위) |
| EUR/KRW 1,612.30 | 1유로 = 1,612.30원 |

> **개발자 참고:** `CurrencyRate` 모델의 `unitLabel`은 통화마다 기준 단위가 다릅니다 (JPY는 100엔 기준). 이것은 관례입니다.

## ETF란?

**ETF (Exchange Traded Fund, 상장지수펀드)**는 여러 종목을 묶어서 하나의 주식처럼 거래할 수 있는 상품입니다.

예: "KODEX 삼성그룹" ETF → 삼성전자, 삼성SDI 등 삼성 계열사 주식을 한 번에 투자

앱에서 검색 화면(`SearchView`)의 ETF 탭에서 국내 ETF와 해외 ETF를 구분해서 보여줍니다.

---

## 분할주(Fractional Shares)

분할주는 **주식 1주를 쪼개서** 소액으로 투자할 수 있는 방식입니다.

예: 삼성전자 1주가 72,000원인데, 10,000원어치만 살 수 있음 (약 0.14주)

앱에서 `StockTagPill`로 "소수점" 태그를 표시하여 분할 매수 가능 종목을 나타냅니다.

---

## 테마(Market Theme)

테마는 **공통 특성을 가진 종목들의 그룹**입니다. 특정 산업이나 이슈에 관련된 종목들이 함께 움직이는 경향이 있습니다.

| 테마 예시 | 관련 종목 |
|-----------|-----------|
| #반도체 | 삼성전자, SK하이닉스, DB하이텍 |
| #2차전지 | LG에너지솔루션, 삼성SDI, 포스코퓨처엠 |
| #AI | 네이버, 카카오, NHN |

앱에서 `MarketTheme` 모델과 `MarketThemeBubbleChartView`(버블 차트)로 테마 트렌드를 시각화합니다. 버블 크기는 테마의 가중치(`weight`), 색상은 상승/하락 방향을 나타냅니다.
