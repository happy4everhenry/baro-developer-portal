# 기술적 분석 (차트)

<PageMeta status="live" layer="shared" />

**기술적 분석**은 과거 가격과 거래량 데이터를 차트로 분석하여 미래 가격을 예측하려는 방법입니다. 앱의 `StockChartView`와 `SubIndicatorChartView`에서 구현됩니다.

---

## 차트 유형 — 전체 17종

`ChartType` enum에 정의된 차트 유형들입니다. `ChartSettingsSheet`에서 사용자가 선택할 수 있습니다.

### 캔들스틱 차트 (Candlestick)

가장 널리 사용되는 차트입니다. 하나의 **캔들(봉)**이 하나의 기간(1일, 1시간 등)을 나타냅니다.

```
    ┃ ← 윗꼬리 (High - 고가)
   ┏┃┓
   ┃ ┃ ← 몸통 (시가~종가)
   ┗┃┛
    ┃ ← 아랫꼬리 (Low - 저가)
```

| 색상 | 조건 | 의미 |
|------|------|------|
| 🔴 빨간 봉 (양봉) | 종가 > 시가 | 가격이 올랐음 |
| 🔵 파란 봉 (음봉) | 종가 < 시가 | 가격이 내렸음 |

- **몸통이 길다** = 가격 변동이 크다 (강한 방향성)
- **꼬리가 길다** = 해당 방향으로 갔다가 되돌아왔다 (반전 신호 가능)

## 차트 기간 (Period)

`ChartPeriod`로 정의됩니다. `StockDetailPeriodTabsView`에서 선택합니다.

| 탭 표시 | 기간 | 데이터 단위 | 용도 |
|---------|------|-------------|------|
| 1일 | 당일 | 분봉 | 단기 트레이딩 |
| 1개월 | 30일 | 일봉 | 단기 추세 |
| 3개월 | 90일 | 일봉 | 중기 추세 |
| 1년 | 365일 | 일봉 | 장기 추세 |
| 3년 | 1,095일 | 주봉 | 장기 투자 판단 |

> **개발자 참고:** LTTB (Largest-Triangle-Three-Buckets) 알고리즘으로 최대 300개 포인트로 다운샘플링합니다. 스크러빙(터치 드래그)할 때는 원본 데이터를 사용합니다.

### 주요 오버레이 상세 설명

#### 이동평균 (Moving Average, MA)

**일정 기간 동안의 평균 가격**을 선으로 표시합니다. 가격의 "소음"을 걸러내고 추세를 파악합니다.

```
MA(20) = 최근 20일간 종가의 평균
```

| MA 종류 | 약어 | 설명 |
|---------|------|------|
| 단순이동평균 | SMA | 단순 산술 평균. 모든 데이터에 동일 가중치 |
| 지수이동평균 | EMA | 최근 데이터에 더 큰 가중치. SMA보다 빠른 반응 |
| 이중지수이동평균 | DEMA | EMA의 EMA. 더 빠른 반응 |
| 삼중지수이동평균 | TEMA | EMA의 EMA의 EMA. 가장 빠른 반응, 지연 최소화 |

**활용:**
- 가격이 MA 위에 있으면 → 상승 추세
- 가격이 MA 아래에 있으면 → 하락 추세
- **골든 크로스:** 단기 MA가 장기 MA를 위로 돌파 → 매수 신호
- **데드 크로스:** 단기 MA가 장기 MA를 아래로 돌파 → 매도 신호

#### 볼린저 밴드 (Bollinger Bands, BB)

MA를 중심으로 **상단/하단 밴드**를 그려 가격의 변동 범위를 표시합니다.

```
상단 밴드 = MA(20) + (표준편차 × 2)
중심선    = MA(20)
하단 밴드 = MA(20) - (표준편차 × 2)
```

- 가격이 **상단 밴드 근처** → 과매수 (overbuying), 하락 가능성
- 가격이 **하단 밴드 근처** → 과매도 (overselling), 반등 가능성
- **밴드 폭이 좁아지면** → 큰 가격 변동이 임박할 수 있음 ("스퀴즈")

#### 일목균형표 (Ichimoku Cloud)

일본에서 개발된 복합 지표로, **5개의 선**과 **구름대**로 구성됩니다.

| 구성 요소 | 계산 | 역할 |
|-----------|------|------|
| 전환선 (Tenkan-sen) | (9일 최고 + 9일 최저) / 2 | 단기 추세 |
| 기준선 (Kijun-sen) | (26일 최고 + 26일 최저) / 2 | 중기 추세 |
| 선행스팬 A (Senkou A) | (전환선 + 기준선) / 2, 26일 선행 | 구름대 상단 |
| 선행스팬 B (Senkou B) | (52일 최고 + 52일 최저) / 2, 26일 선행 | 구름대 하단 |
| 후행스팬 (Chikou) | 현재 종가를 26일 후행 표시 | 과거 대비 현재 위치 |
