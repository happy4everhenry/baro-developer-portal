# Market Overview Screen

This is a full-screen detail page that shows you all the market indices (like KOSPI, NASDAQ, S&P 500) and currency exchange rates (like USD, EUR, JPY) in one place. You get here by tapping the "more" arrow on the Home screen's market overview row. It is a simple two-tab list for browsing financial data.

    ### API Endpoint Mapping

    | Section | Repository Method | REST Endpoint | Response Model | Fallback |
    |---|---|---|---|---|
    | MO2 Indices | `repository.fetchAllIndices()` | `GET /api/market/indices` | `[MarketIndex]` | Error alert + retry button |
    | MO3 FX Rates | `repository.fetchCurrencyRates()` | `GET /api/market/currency-rates` | `[CurrencyRate]` | Error alert + retry (if no prior error) |

    <InfoBox type="warning" title="Sequential Loading">
      Unlike the Home screen, this screen loads indices first, then currency rates -- sequentially, not in parallel. Each can fail independently without blocking the other.

    ### View State Properties

    | Property | Type | Default | Description |
    |---|---|---|---|
    | `selectedTab` | `MarketOverviewTab` | `.indices` | Controls which data list is rendered |
    | `allIndices` | `[MarketIndex]` | `[]` | Full list of market indices from API |
    | `allRates` | `[CurrencyRate]` | `[]` | Full list of currency exchange rates from API |
    | `isLoadingIndices` | `Bool` | `true` | Shows skeleton rows when true and data is empty |
    | `isLoadingRates` | `Bool` | `true` | Shows skeleton rows when true and data is empty |
    | `errorMessage` | `String?` | `nil` | Korean error message for alert body |
    | `showError` | `Bool` | `false` | Controls error alert presentation |

    ### State Machine

    **Initial Load (per data source):** Idle (isLoading=true, data=[]) -> Skeleton (10 rows) -> Loaded (data populated)

    **Error Recovery:** Loading -> Error Alert ("데이터 로딩 실패") -> User taps "재시도" -> loadInitialData() -> Loaded

    **Tab Switch:** Tab A Visible -> Tap Tab B -> scrollTo("scrollTop") -> Tab B Visible

    | Section | Skeleton View | Error Behavior | Skeleton Condition |
    |---|---|---|---|
    | MO1 Tab Header | None (static) | N/A | Always rendered |
    | MO2 Indices List | `DetailRowSkeletonView` x 10 | Alert: "지수 데이터를 불러올 수 없습니다." | `isLoadingIndices && allIndices.isEmpty` |
    | MO3 FX Rates List | `DetailRowSkeletonView` x 10 | Alert: "환율 데이터를 불러올 수 없습니다." (only if no prior error) | `isLoadingRates && allRates.isEmpty` |

    ### Error Handling

    | Section | Strategy | Error Message (Korean) |
    |---|---|---|
    | MO2 Indices | Alert with retry | "지수 데이터를 불러올 수 없습니다." |
    | MO3 FX Rates | Alert with retry (suppressed if prior error) | "환율 데이터를 불러올 수 없습니다." |

    <Modal triggerLabel="View Data Loading Code" title="MarketOverviewDetailView.swift -- Data Loading" size="medium">

    ```swift
    // Indices load first, then currency rates -- each fails independently
    private func loadInitialData() async {
        do {
            allIndices = try await repository.fetchAllIndices()
        } catch {
            errorMessage = "지수 데이터를 불러올 수 없습니다."
            showError = true
        }
        isLoadingIndices = false

        do {
            allRates = try await repository.fetchCurrencyRates()
        } catch {
            if errorMessage == nil {
                errorMessage = "환율 데이터를 불러올 수 없습니다."
                showError = true
            }
        }
        isLoadingRates = false
    }

    // Triggered once on view appear
    .task { await loadInitialData() }
    ```
