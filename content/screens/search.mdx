# Search Screen

The Search screen lets you find any stock, ETF, or market theme in the app. It has five tab categories, a floating search bar at the bottom, and instant autocomplete suggestions. All stock and theme data is pre-loaded when the screen opens so that searching feels instant -- no waiting for the server on every keystroke.

    ### How Data Loads

    <StepGuide>
      <Step number={1} title="View Appears">
        `.task` fires: search field auto-focuses via `@FocusState`, then calls `loadAllStocksIfNeeded()`.
      </Step>
      <Step number={2} title="Parallel Fetch">
        Three `async let` calls fire at the same time: domestic stocks, US stocks, and market themes.
      </Step>
      <Step number={3} title="Cache">
        Results stored in `allDomesticStocks`, `allUSStocks`, `allThemes`.
      </Step>
      <Step number={4} title="Guard">
        `hasLoadedAllStocks = true` prevents making the same network calls again.
      </Step>
      <Step number={5} title="Search Ready">
        All filtering is instant via computed properties reading the cached arrays. No more network calls needed.
      </Step>
    </StepGuide>

    <InfoBox type="tip" title="Prefix Matches First">
      The autocomplete system shows up to 3 suggestions. It prioritizes names that start with your search text over names that merely contain it. This makes the suggestions feel smarter.
    </InfoBox>

    ### Search Algorithm

    <InfoBox type="info" title="Client-Side Filtering">
      All search filtering runs on the device against pre-cached arrays. No server-side search API calls happen during typing. The autocomplete prioritizes prefix matches over substring matches, returning max 3 suggestions.
    </InfoBox>

    <Modal triggerLabel="View Search Algorithm Code" title="SearchViewModel.swift -- Filtering & Autocomplete" size="large">

    ```swift
    // Client-side filtering -- runs against cached arrays for instant results
    private func filterByQuery(_ stocks: [Stock]) -> [Stock] {
        guard !searchQuery.isEmpty else { return [] }
        let query = searchQuery.lowercased()
        return stocks.filter {
            $0.name.lowercased().contains(query) ||
            $0.code.lowercased().contains(query)
        }
    }

    // Computed properties for each category
    var searchDomesticResults: [Stock] { filterByQuery(allDomesticStocks) }
    var searchUSResults: [Stock]       { filterByQuery(allUSStocks) }

    var searchThemeResults: [MarketTheme] {
        guard !searchQuery.isEmpty else { return [] }
        let query = searchQuery.lowercased()
        return allThemes.filter {
            $0.name.lowercased().contains(query) ||
            $0.description.lowercased().contains(query)
        }
    }
    ```

    ```swift
    /// Top 3 names matching the current query (stocks + themes)
    /// Prefix matches are prioritized over substring matches
    var autocompleteSuggestions: [String] {
        guard !searchQuery.isEmpty else { return [] }
        let query = searchQuery.lowercased()

        var seen = Set<String>()
        var prefixMatches: [String] = []
        var substringMatches: [String] = []

        for stock in (allDomesticStocks + allUSStocks) {
            let name = stock.name
            guard !seen.contains(name) else { continue }
            let lower = name.lowercased()
            if lower.hasPrefix(query) {
                prefixMatches.append(name); seen.insert(name)
            } else if lower.contains(query) {
                substringMatches.append(name); seen.insert(name)
            }
        }

        for theme in allThemes {
            let name = theme.name
            guard !seen.contains(name) else { continue }
            let lower = name.lowercased()
            if lower.hasPrefix(query) {
                prefixMatches.append(name); seen.insert(name)
            } else if lower.contains(query) {
                substringMatches.append(name); seen.insert(name)
            }
        }

        return Array((prefixMatches + substringMatches).prefix(3))
    }
    ```

    ### State Machine

    **Initial Load:** View Appears -> Focus Search Bar -> loadAllStocksIfNeeded() -> All Data Cached

    **Search Query:** Idle (최근검색) -> User Types -> Tab label becomes "검색결과" -> Cross-Category Results

    **Tab Switch:** Tap Tab -> selectTab() -> loadStocks() -> Skeleton (15 rows) -> Data Rendered

    **Empty State:** Query Input -> Filter All Categories -> All Empty + hasLoaded -> "검색 결과가 없습니다"

    | Section | Skeleton View | Error Behavior | Empty Behavior |
    |---|---|---|---|
    | SR1 Search Bar | None (always static) | N/A | Always rendered with placeholder |
    | SR2 Filter Tabs | None (always static) | N/A | First tab label changes dynamically |
    | SR3 Recent Searches | None | N/A | Hardcoded list (currently 1 item) |
    | SR4 Search Results | `SearchRowSkeletonView` x15 + `.shimmer()` | Silent fail (print only) | "검색 결과가 없습니다" centered |
    | SR5 NXT Toggle | None (always static) | N/A | Only on 국내주식 tab |

    <Modal triggerLabel="View Data Loading Code" title="SearchViewModel.swift -- Parallel Pre-Loading & Per-Tab Loading" size="large">

    ```swift
    /// Load all data sources in parallel for instant cross-category search.
    func loadAllStocksIfNeeded() async {
        guard !hasLoadedAllStocks else { return }
        do {
            async let domesticTask = marketRepository.fetchSearchStocks(domestic: true)
            async let usTask = marketRepository.fetchSearchStocks(domestic: false)
            async let themesTask = marketRepository.fetchMarketThemes()
            allDomesticStocks = try await domesticTask
            allUSStocks = try await usTask
            allThemes = try await themesTask
            hasLoadedAllStocks = true
        } catch {
            print("[Search] Failed to load all stocks: \(error)")
        }
    }
    ```

    ```swift
    /// Per-tab data loading with skeleton shimmer during fetch.
    func loadStocks() async {
        isLoading = true
        do {
            switch selectedTab {
            case .recent:   stocks = []
            case .domestic: stocks = try await marketRepository.fetchSearchStocks(domestic: true)
            case .us:       stocks = try await marketRepository.fetchSearchStocks(domestic: false)
            case .etf:
                let domestic = try await marketRepository.fetchSearchStocks(domestic: true)
                let foreign = try await marketRepository.fetchSearchStocks(domestic: false)
                stocks = domestic + foreign
            case .theme:
                stocks = []
                if allThemes.isEmpty {
                    allThemes = try await marketRepository.fetchMarketThemes()
                }
            }
        } catch {
            print("[Search] Failed to load stocks: \(error)")
        }
        isLoading = false
    }
    ```
