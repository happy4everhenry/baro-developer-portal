# All Screens

<InfoBox type="fun" title="What's a screen?">
A **screen** is like a page in a book. When you open BARO, you see one screen at a time. You tap buttons to flip to a different screen — just like turning pages! BARO has **13 screens**, and each one has a special job.
</InfoBox>

## What Every Screen Has

Every screen in BARO follows the same pattern:

<StepGuide>
  <Step number={1} title="A View (the pretty part)">
    This is the SwiftUI code that draws everything you see — buttons, text, charts, and colors.
  </Step>
  <Step number={2} title="A ViewModel (the brain)">
    This is the code that decides *what* to show. It grabs data and gets it ready for the View.
  </Step>
  <Step number={3} title="A Repository (the helper)">
    This fetches data from the internet or from saved files on your phone.
  </Step>
</StepGuide>
